# Bioinfo_sh_Scripts
## Bash Scripts for Bioinformatics
## Linux Commands
Display the message "Hello, World!"
```bash
$ echo Hello, World!
```
Print Working Directory
```bash
$ pwd
```
Current Directory
```bash
$ cd .
```
Parent Directory
```bash
$ cd .. 
```
Change Directory
```bash
$ cd /path/to/the/new/directory
```
